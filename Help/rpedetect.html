Keyboard Shortcuts and Mouse Operations

Key Action Description
F1  Keyboard Shortcuts... Display this help screen.
F2 Toggle Visibility Temporarily switch off/on marker visibility. Visibility is automatically restored when you open images or detect RPE cells.
F10 Reset Image View In the “Default” mouse mode dragging the mouse changes brighness/contrast of the image. Pressing F10 reverts these changes and resets the image display to default settings. Also helpful if you zoom in/out too much or accidentally rotate the image (with Ctrl+left-dragging), and you want the origional view back.
Ctrl+I Toggle Interpolation Togle pixel interpolation for image zoom. If the interpolation is on, a bicubic resampling is used. Otherwise, nearest neighbor.
Ctrl+O  Open Images... Open one or more AO Images (.tif) and associated Annotations (.csv).
   Open Annotations... Open Annotations (.csv). Annotation data is applied to open images with matching file name
Ctrl+S  Save Annotations... Save RPE detection results of all open images.
  Delete Annotations Delete all RPE cell markers (annotations) on the current image.
Ctrl+L Show data file locations Displays the “Image Data Location” box showing full paths to the currently displayed image, its associated contour spreadsheet in the same directory, and auto-backup contour spreadsheet. The auto-backup (history) file is updated after each contour operation, and it is loaded when there is no manually saved spreadsheet in the image file directory. It prevents your editings from being lost when you accidentally close the program (or open new images) without saving the data.
Ctrl+G  Detect Display RPE Detection dialog. On this dialog you can adjust detection parameters and choose one or more open images to detect RPE cells on. Note that clicking on the little checkmark √ in the table header checks/unchecks all images in the list.
Ctrl+M  Default Switch to “Default” mouse mode. In this mode:
– Mouse wheel (or Ctrl+right-dragging) zooms in and out;
– Dragging the mouse changes brightness/contrast;
– Shift+left-dragging (or middle-dragging) scrolls the image.
– Ctrl+left-dragging (Windows) rotates the image.
Ctrl+C  Draw Switch to “Add Markers” mouse mode. In this mode:
– Clicking mouse on the image adds a new marker;
– Mouse wheel (or Ctrl+right-dragging) zooms in and out;
– Shift+left-dragging (or middle-dragging) scrolls the image.
– Ctrl+left click (Windows) or Alt+left click (Mac OS X) on a marker erases the marker.
Ctrl+E  Move Switch to “Move Markers” mouse mode. In this mode:
– Clicking on a marker and dragging the mouse moves the marker;
– Mouse wheel (or Ctrl+right-dragging) zooms in and out;
– Ctrl+left click (Windows) or Alt+left click (Mac OS X) on a marker erases the marker.
– Shift+left-dragging (or middle-dragging) scrolls the image.
Ctrl+D  Erase M Switch to “Erase Multiple Cone Contours” mouse mode. In this mode:
– Left-dragging the mouse around an area erases all markers inside this area;
– Mouse wheel (or Ctrl+right dragging) zooms in and out;
– Shift+left-dragging (or middle-dragging) scrolls the image.
Ctrl+W  Erase S Switch to “Erase Single RPE Cell Marker” mouse mode. In this mode:
– Left-clicking on a marker erases it;
– Mouse wheel (or Ctrl+right-dragging) zooms in and out;
– Shift+left-dragging (or middle-dragging) scrolls the image.
Ctrl+Z  Undo Undo the last “Add”, “Move” or “Erase” operation.
Ctrl+V  Voronoi Toggle display of the Voronoi Diagram on contour center points.
